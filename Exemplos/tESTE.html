Oque é HTML? Linguagem de marcação de texto

Principais tags de html.

Todas as tags são usadas "(< >)" abrindo e da mesmo forma fechando so que deverá colocar uma (/) ápos o primeiro MENOR.

Por Exemplo:


div: Usadas para agrupar blocos de conteúdo e aplicar estilos ou scripts.
span:Usada para agrupar conteúdo em linha e aplicar estilos ou scripts.
ul: Criar uma lista não ordenada (pontos).
ol: Criar uma lista ordenada(números).
li: Define um item de lista usado dentro do UL ou Ol.
table:Criar uma tabela.
tr: Define uma linha de tabela, usada dentro de TABLE.
td: Define uma célula de tabela, usada dentro de TR.
th: Define uma célula de cabeçalho em uma tabela.
form: Criar um formulário para coletar dados de usuários.
input:Define um campo de entrada dentro de um formulário.
buttom: Criar um botão clicável.
a: a href="link"
img: img src="link da imagem puxada de uma pasta" alt= descrição de imagem
strong ou b: Usado para fonte ficar em negrito.
i:Usado para fonte ficar itálico

Indetificadores

id:É usado para identificar um único elemento da página.
class: É usado para identificar um ou mais elementos que pertencem a um mesmo grupo ou categoria.
